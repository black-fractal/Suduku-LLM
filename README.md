# Suduku-LLM

Interactive Sudoku (generator + solver + Tkinter GUI).

Run the GUI:

```bash
python run.py
```

Run tests (requires pytest):

```bash
pytest -q
```

Files of interest:
- `sudoku/core.py` - generator and backtracking solver
- `sudoku/gui.py` - simple Tkinter GUI
- `run.py` - launcher

Enjoy!
